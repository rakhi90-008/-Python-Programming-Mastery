# Python Personal Finance Manager

## Project Overview
The Python Personal Finance Manager is a command-line based application developed using Python.  
It helps users track daily expenses, store them persistently using CSV files, and generate simple financial reports.

This project is designed to demonstrate:
- Python programming fundamentals
- Object-Oriented Programming (OOP)
- File handling and data persistence
- Error handling and input validation
- Modular code organization
- Command-line interface (CLI) development

---

## Project Objectives
- Track and manage personal expenses
- Store expense data persistently using CSV
- Generate category-wise and monthly reports
- Apply OOP concepts in a real-world project
- Practice clean code organization and documentation

---

## Technology Stack
- **Programming Language:** Python 3
- **Interface:** Command Line (CLI)
- **Data Storage:** CSV File
- **IDE:** VS Code
- **Libraries Used:** csv, datetime, collections

---

## Features
- Add new expense
- View all expenses
- Category-wise expense summary
- Monthly expense report
- Input validation and error handling
- CSV-based data persistence
- Modular project structure
